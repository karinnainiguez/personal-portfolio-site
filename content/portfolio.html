Software Development Portfolio

Below are some examples of the projects I have completed. Feel free to click on any of them to see the details in my GitHub page.

  • Calculator

    A terminal program with a command line interface. Using user input, it performs simple arithmetic.

  • Solar System

    Model of a solar system using Ruby classes.

  • Word Guess

    A Ruby word game. Through the terminal, it allows users to guess letters in a word and has an ASCII art as feedback on their progress in the game.

  • Grocery Store

    Ruby program using classes and inheritance. It simulates a grocery store system that tracks both in-person and online orders.

  • Scrabble

    Project using TDD and pair programming to develop an app that allows users to play words for points based on letter scores and bonuses.

  • Ride Share

    A design exercise to create a tracking system for a ride share application. This includes use of CSV files, classes, and object-orientation.

  • Hotel

    Ruby system using object-orientation, with classes, methods, and design. The application is a booking system for a small hotel.
